## ABSOLUTE COLOR RULES

### DO:
- Use gold for primary CTAs: "Get Free Quote", "Call Now"
- Use red ONLY for genuine emergency/urgent messaging
- Maintain charcoal as primary background for brand strength
- Create high contrast (gold on charcoal, white on charcoal)
- Use white generously as breathing space

### DON'T:
- Introduce ANY colors not listed above (no blues, greens, oranges, etc.)
- Use gold on white backgrounds (poor contrast)
- Use red for non-emergency content
- Mix gold and red in close proximity (visual conflict)
- Use more than 10% combined gold + red in one composition

### Accessibility Standards

All color combinations must meet **WCAG AA** standards minimum:

- Gold text on charcoal: **Pass** (4.8:1 ratio)
- White text on charcoal: **Pass** (15.2:1 ratio)
- Red text on white: **Pass** (5.9:1 ratio)
- White text on red: **Pass** (5.9:1 ratio)
- Gold text on white: **Fail** (1.9:1 ratio) â€” NEVER USE

## TYPOGRAPHY SYSTEM

### Font Strategy

**Core Principle:** Clean, modern, highly legible. Professional without being cold.

### Primary Typeface: Inter

| Style | Weight | Size Range | Use Case |
|-------|--------|------------|----------|
| Inter Bold | 700 | 32-64px | Hero headlines, major CTAs |
| Inter Semi-Bold | 600 | 24-32px | Section headers, subheads |
| Inter Medium | 500 | 18-24px | Card titles, emphasized text |
| Inter Regular | 400 | 16-18px | Body copy, descriptions |

**Why Inter:**
- Designed for screens (high legibility)
- Professional and modern
- Extensive character set (includes Spanish accents)
- Free and web-safe
- Neutral without being generic

### Typography Scale

```
DESKTOP/PRINT:
H1 â€” Hero Headlines: 48-64px / Bold / 110% line height
H2 â€” Page Headers: 36-48px / Semi-Bold / 120% line height
```

### Typography Pairing Rules

#### DO:
- Use Inter for ALL body copy and UI text
- Maintain consistent line heights (never tighter than 110%)
- Use bold weights for emphasis, not underline
- Ensure minimum 16px for body text (accessibility)
- Use charcoal (#1c191a) for primary text
- Use medium gray (#6b7280) for secondary text only

#### DON'T:
- Mix multiple font families
- Use script, decorative, or serif fonts
- Use all-caps for body text (headlines okay)
- Use tight leading (line height) that reduces readability
- Use text smaller than 14px for any user-facing content

## PHOTOGRAPHY DIRECTION

### Core Photography Principles

1. **Authentic Over Staged** â€” Real work, real homes, real team
2. **Natural Lighting** â€” Soft, diffused, realistic (not dramatic)
3. **Charlotte Context** â€” Recognizable neighborhoods and homes
4. **Human Connection** â€” Owners and team visible, approachable
5. **Action Over Posing** â€” Hands working, problems being solved

### Required Photo Categories

| Category | Description | Quantity Needed |
|----------|-------------|-----------------|
| Owner Portraits | Jaime & Hamed, approachable, professional | 6-10 variations |
| Team at Work | Technicians diagnosing, fixing, explaining | 20+ images |
| Before/After | Transformations showing results | 10+ pairs |
| Hands-On Details | Close-ups of work being done | 15+ images |
| Client Interaction | Team explaining solutions, client relief | 8-12 images |
| Charlotte Homes | Exterior and interior of local homes | 10+ images |
| Emergency Response | Team arriving, assessing urgent situations | 6-10 images |

### Photography Style Guidelines

**COMPOSITION:**
- Rule of thirds (avoid centering)
- Clean backgrounds (declutter before shooting)
- Include brand elements when possible (truck, tools, uniforms)
- Show context (recognizable Charlotte homes)

**LIGHTING:**
- Natural light preferred (soft, diffused)
- No harsh shadows or dramatic lighting
- Avoid overly dark or overly bright exposures
- Consistent color temperature (warm, not cool)

**SUBJECTS:**
- Owners (Jaime & Hamed) should be prominent
- Show faces when possible (builds trust)
- Action shots over posed portraits
- Include diversity of projects and homes

**COLOR GRADING:**
- Natural, realistic tones (no heavy filters)
- Slight warmth okay (inviting, not cold)
- Maintain accurate material colors
- Don't oversaturate or apply Instagram filters

### Photo DO's and DON'Ts

#### DO:
- Show real work in progress (authentic)
- Include owner presence (trust-building)
- Capture client reactions (relief, satisfaction)
- Show before/after transformations
- Include recognizable Charlotte neighborhoods
- Use natural lighting and realistic colors
- Show close-ups of quality workmanship

#### DON'T:
- Use generic stock photos (fake, corporate feeling)
- Show luxury staged interiors (not on-brand)
- Hide faces or show only backs (reduces trust)
- Use dramatic/cinematic lighting (too polished)
- Show disasters or damage without context
- Include competitor branding or logos
- Use heavily filtered or stylized images

### Photo Treatment & Overlays

When applying brand elements to photography:

**1. Logo Placement:**
- Use white/reverse logo on photos
- Apply 70% opacity charcoal rectangle behind logo
- Place in bottom-right or top-left corner
- Maintain full clear space requirements

**2. Text Overlays:**
- Use white text with subtle charcoal shadow/backdrop
- Keep text minimal and large (readability)
- Avoid busy areas of photo for text placement
- Use bold weight for photo overlays

**3. Color Overlays:**
- If needed, use 20-30% opacity charcoal overlay
- Helps with text legibility on busy photos
- Never tint photos gold or red (unnatural)

## ICONOGRAPHY & GRAPHICS

### Icon Style Specifications

- **Style:** Line icons (outline style, not filled)
- **Stroke Weight:** 2px
- **Corner Radius:** 4px (rounded, not sharp)
- **Color:** Charcoal (#1c191a) on light backgrounds, white (#ffffff) on dark
- **Grid:** 24x24px base, scalable to 32px, 48px

### Core Icon Set

**SERVICES:**
- Wrench (general handyman)
- Water droplet (plumbing)
- Lightning bolt (electrical)
- Paintbrush (painting/drywall)
- Door (door/window repair)
- House with tools (home repair general)
- Toolbox (multiple services)
- Alert triangle (emergency services)

**COMMUNICATION:**
- Phone (call us)
- Calendar (schedule appointment)
- Clock (fast response, 24/7)
- Chat bubble (quote request)
- Star (reviews, quality)
- Map pin (local service area)
- Checkmark (job done right)
- People (team, owners)

### Icon Usage Rules

#### DO:
- Use 2px stroke weight consistently
- Maintain 24px base size (scale proportionally)
- Use charcoal on light, white on dark
- Align icons to text baseline
- Use generous spacing between icon and text (8-12px)

#### DON'T:
- Use filled/solid icons (conflicts with line style)
- Mix icon styles (all must be line-based)
- Use decorative or playful icons (stay professional)
- Color icons gold or red (use brand color system sparingly)
- Use icons smaller than 20px (legibility issues)
